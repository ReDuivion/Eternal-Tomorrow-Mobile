## Installation

Install my-project with npm

```bash
  npm install 
  cd L4TOMO
  npm run android || npm run ios
```

Dont Forget To Setup Android Studio By Yourself
    
## Tech Stack

**Client:** React, React Native, MUI React Native Paper

**Server:** NodeJs, Expo Router, Supabase
